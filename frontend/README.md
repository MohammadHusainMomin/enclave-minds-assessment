# Enclave Minds – MERN Technical Assessment

This repository contains my solutions for the Enclave Minds Internship technical assessment, implemented using the MERN stack.

## Tech Stack
- Frontend: React (Vite), JavaScript, Context API, CSS
- Backend: Node.js, Express.js, JWT, MongoDB (Mongoose)

## Implemented Tasks
- React dynamic user search (real-time, no UI flicker)
- Secure POST API with email & password validation
- Mongoose schema for Task Management
- JWT authentication middleware + frontend token attachment
- Multi-step registration form with preserved state
- Global error handling for 404 & 500 responses
- MongoDB aggregation for revenue by category

## Run Locally

### Frontend
cd frontend  
npm install  
npm run dev  

### Backend
cd backend  
npm install  
npm start
